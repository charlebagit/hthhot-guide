# hthhot-guide

## 项目简介

`hthhot-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名、网站或服务，仅作为静态页面集合的存储与展示空间。

所有页面均为独立 HTML 文件，可直接通过浏览器打开，无需后端支持。

---

## 目录说明

```
hthhot-guide/
├── index.html          # 主入口页面
├── pages/              # 存放各个独立 HTML 页面的文件夹
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：可选的导航或汇总页面，用于快速跳转至各子页面。
- `pages/`：所有独立 HTML 归档页面均放置于此目录下，文件名可反映内容主题。

---

## 页面归档说明

- 每个 HTML 页面均为自包含文件，样式、脚本均内嵌或使用相对路径引用。
- 页面内容可能涉及信息整理、工具说明、资料展示等，具体以文件实际内容为准。
- 本仓库不保证页面内容的时效性或完整性，请以页面自身标注的信息为准。

---

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时会直接提交至 `main` 分支。
- 欢迎通过 Issue 或 Pull Request 提交建议或修正，但请注意不要引入具体域名或营销内容。
- 所有提交均需保持内容中立、客观，符合通用归档目的。

---

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用任意浏览器打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务直接部署本仓库。
